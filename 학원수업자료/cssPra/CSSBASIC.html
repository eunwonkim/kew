CSS(Cascading Style Sheets) - HTML, XML등 구조화문서를 화면 종이등에 어떻게 그릴것인지(렌더링)정의하는 언어. - CSS는 HTML 각 요소의 스타일을 정의하여 화면에 어떻게 렌더링 하면 되는지 브라우저에게 설명해주는 언어. CSS 관련 용어 1. 셀렉터(선택자) - css -> html 요소의 스타일정의 -> 스타일을 적용하고자 하는 HTML요소를 선택 - 셀렉터는 스타일을 적용하고자 하는 html 요소를 선택하기위해 css에서 제공하는 수단. - 선택자 종류 1) html 선택자 - 태그명 그 자체를 작성 2) id 선택자 - #id명 3) class 선택자 - .class명 4) 그룹선택자 - 여러개의 태그, id, 클래스 작성후 ,로 구분 셀렉터 프로퍼티(속성명) 프로퍼티(속성값) p { background-color : red; width : 500px; } 2. 프로퍼티명(속성명) - 셀렉터로 HTML요소를 선택하고 블럭 내에 프로퍼티 값을 지정하는 것으로 스타일을 정의할수 있다. -> 이미 지정된 프로퍼티명을 사용해야한다. 여러개의 프로퍼티를 연속으로 지정할수 있다(세미콜론으로 구분.) 3. CSS의 적용 1) 인라인 선택(인라인 스탈 / 인라인 시트) - HTML 태그내에 style을 부여하는 방법 - 해당 태그에만 스타일을 부여할수 있고 꼭 필요한 경우에만 사용하는것을 권장 (절대적으로 변경되면 안돼는 스타일 적용시.) 2) 내부 선택(내부스타일 / 내부 시트) embedding style - html head 영역에 style 이라는 태그를 선언후 스타일을 부여하는 방법 - html과 css는 서로 역할이 다르기 때문에 대부분은 다른 파일로 구분되어 작성하고 관리하는것이 유지보수상 바람직. 3) 외부 선택(외부 스타일 / 외부 시트) link style - .css라는 확장자를 가진 파일을 링크해서 하나의 파일에서 스타일을 부여하는법 스타일 적용의 우선순위 인라인 > 내부 > 외부 > 기본
