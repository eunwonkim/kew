객체 : 자바스크립트를 이루고 있는 거의 모든것. - 원시타입을 제외한 나머지 값들.(함수, 배열, HTML, CSS) 자바스크립트의 객체는 key, value로 구성된 프로퍼티(속성)의 집합. - 프로퍼티의 값으로 JS에서 사용할수 있는 모든 값을 사용할수 있다. -> js에서의 함수가 1급 객체이므로. -> 프로퍼티의 값이 함수일경우 메서드라는 호칭으로 부른다. 결론 : JS에서의 객체는 데이터를 의미하는 프로퍼티와 데이터를 참조하고 조작할수 있는 메서드로 구성된 집합. 객체는 프로퍼티와 메서드를 모두 포함할수 있어서. 둘을 하나의 단위로 구조화 할수 있다. 프로토타입 : 객체지향 상속을 구현하기위해 프로토타입이라 불리는 다른객체의 프로퍼티와 메서드를 상속받는 개념. 프로퍼티 : key, value로 구성되며 프로퍼티는 프로퍼티 키로 식별. 1) 프로퍼티 작성시 규칙 - key : 모든 문자열 - value : 모든 값 2) 프로퍼티는 순서를 보장하지 않는다. 3) 이미 존재하는 프로퍼티 키를 중복선언하면 나중에 선언한 프로퍼티가 먼저 선언한 프로퍼티를 덮어 씌움.
